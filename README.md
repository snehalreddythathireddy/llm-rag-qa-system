# LLM-powered RAG Question Answering System

This project implements a Retrieval-Augmented Generation (RAG) system that allows users to upload PDF documents and ask questions based on the content.

## Features
- Upload PDF documents
- Ask questions from uploaded files
- Semantic search using FAISS
- Free Hugging Face models
- Simple Streamlit UI

## Tech Stack
- Python
- LangChain
- Hugging Face Transformers
- FAISS
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
